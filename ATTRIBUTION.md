# Attribution

This project gives full structural credit to the original Piconomix FW Library,
also known as `px-fwlib`.

`nugget-fwlib` was organized with admiration for the way Piconomix FW-LIB
separates embedded firmware into clear, reusable library families: architecture
support, board profiles, GPIO, timing, communications, data handling, devices,
diagnostics, examples, and documentation. That structure made the original
Piconomix project a strong educational model for building firmware that is
readable, modular, and portable across small microcontroller projects.

The original project is:

- Project: Piconomix FW Library (`px-fwlib`)
- Website: https://piconomix.com/px-fwlib/
- GitHub: https://github.com/piconomix/px-fwlib

All credit for that firmware-library organization belongs to Piconomix and the
author or authors of `px-fwlib`.

This repository is an independent Great Cow BASIC teaching library for the
Nugget / REDBOARD / PIC16F18424 pathway. It is not an official Piconomix
project, is not affiliated with Piconomix, and does not claim ownership of the
ideas behind the original FW-LIB structure. Where this repository follows a
similar shape, it does so intentionally as an attribution and tribute to the
clarity of the original Piconomix FW-LIB design.
